# MVC Thang May

## Mô tả

Dự án quản lý thang máy sử dụng NestJS framework với kiến trúc MVC.

## Cài đặt dự án

```bash
$ yarn install
```

## Biên dịch và chạy dự án

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Chạy tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
